# Contributing to plugin-transport-quic-bridge

## Prerequisites

- Go 1.23+
- git

## Development

```bash
# Build
go build ./cmd/

# Test
go test ./... -v

# Vet
go vet ./...
```

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests: `go test ./...`
5. Submit a pull request
